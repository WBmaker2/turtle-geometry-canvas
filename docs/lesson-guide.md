# Turtle Geometry Canvas 수업 활용 안내

## 대상
- 초등학교 5~6학년
- 수학(다각형) 단원: 다각형의 내각의 합, 정다각형의 성질
- 실과(소프트웨어): 순차, 선택, 반복 구조를 적용한 프로그래밍

## 성취기준
- [6수02-04] 다각형의 내각의 크기의 합을 구하고, 정다각형의 성질을 이해한다.  
- [6실05-02] 문제를 해결하기 위한 프로그램을 설계하고, 순차, 선택, 반복 등의 구조를 적용하여 프로그래밍한다.

## 40분 수업 흐름(5단계)

1. 도입 (약 8분)
   - 목표 제시: “거북이를 돌리면 왜 바깥각이 변하나, 안쪽 각은 어떻게 정해질까?”
   - 교사 시연: 기본 블록(이동, 회전, 실행, 초기화) 소개.

2. 탐색 1 (약 12분)
   - 학생이 샘플 과제(삼각형/사각형/오각형) 하나를 불러오고 그대로 실행.
   - 선분 개수와 그려진 모양을 관찰해 정다각형인지 확인.
   - 외부에서 질문: “한 바퀴를 몇 번 꺾었을까?”

3. 탐색 2 (약 10분)
   - 회전각을 바꿔 실험하고 닫힌 도형 여부를 비교.
   - `외각 = 360 / 변의 개수` 개념을 예측해 적어보고, 실행 결과와 대조.
   - 색상 블록으로 선분 구분해 회전 구간을 구분.

4. 정리 활동 (약 6분)
   - `내각 = 180 - 외각`, `내각의 합 = (변 수-2)×180` 정리.
   - 과제별로 계산값과 실행 결과가 일치하는지 토론.

5. 성찰 및 확장 (약 4분)
   - 소수 교대로 오늘의 관찰 포인트 발표.
   - “코드가 바뀌면 결과가 어떻게 바뀌는지”를 한 문장 기록.

## 교사 관찰 포인트
- 학생이 실행 전에 각도/명령 수를 예측해 보는가.
- 실행 후 결과와 수치 계산값을 비교해 오류를 스스로 찾아 고치려는가.
- 과도한 반쪽만 맞는 답(예: 내각만 외우고 외각을 설명 못함)이 아닌, 합계 규칙까지 연결하는가.
- 모바일/태블릿에서도 버튼이 잘 눌리고, 강의 화면에서 줄바꿈이 깨지지 않는가.

## 학생 질문 예시
- “회전 각도를 두 배로 하면 왜 도형이 안 맞을까요?”
- “왜 선분 수가 변의 수와 같아 보이나요?”
- “외각이 120도면, 한 바퀴는 어떻게 360으로 연결되나요?”
- “내각의 합이 일정하지 않은데 어떻게 맞는지 계산해 보면 알 수 있을까요?”
